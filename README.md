# 🌐 Front-End Web Foundations & UI Architecture

**Status:** 🟢 Completed | **Focus:** Responsive Design, SPA Routing, and UI Layouts
**Tech Stack:** HTML5, CSS3, Bootstrap 4, DOM Manipulation (CCBP UI Kit)

This repository serves as a master collection of responsive, static web applications developed during the **CCBP Tech 4.0** Developer Foundations sprint. The primary objective of these builds was to master modern CSS layout architectures (Flexbox/Grid), Bootstrap utility classes, and single-page application (SPA) visual routing before transitioning into heavy backend and AI engineering.

## ⚡ Technical Takeaways
* **Strict CSS Modeling:** Replaced hacky `<br>` tags and fixed heights with fluid `min-height: 100vh` and organic container padding.
* **Component Isolation:** Successfully isolated multiple Bootstrap Carousels on a single DOM tree using unique `data-target` IDs.
* **Version Control:** Maintained a clean, feature-based Git commit history (`feat:`, `docs:`) aligned with professional software engineering standards.
